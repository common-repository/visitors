== Description ==

This plugin is a live hit counter, which displays the number of visitors currently viewing the current page, and the ones currently viewing one page of the current WordPress or Buddy Press site.

It is a Widget plugin (WPW) so that you can drag and drop it anywhere on your site.

== Requirements ==
* php5.3
* WP 2.x
